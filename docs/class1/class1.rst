Application Infrastructure Protection (AIP) 201: Getting started with Cloud Workload Protection
================================================================

Experience a hands-on introduction to Application Infrastructure Protection (AIP), a modern security monitoring solution. Gain knowledge through executing terminal commands against the infrastructure, resulting in real-time visibility into cloud workloads. Additionally, rapidly deploy and enable out-of-box use cases that focus on runtime activity, data loss identification, early identification of Advanced Persistent Threats, and compliance issues.
